# Sales-Profit-Analysis-Dashboard-Tableau-Sample-Superstore-

📊 I recently built an interactive Tableau dashboard to analyze sales performance using the Sample Superstore dataset.
The goal of this project was to transform raw sales data into meaningful insights through data visualization and interactive dashboard design.
The dashboard explores:
• Sales, Profit, and Orders performance
• Sales distribution by category and customer segment
• Regional sales performance across the United States and Canada
• Top-performing products and sub-categories
• Delivery performance and average shipping times
Key insights from the analysis:
• Technology category generates the highest sales performance
• The West region leads total sales contribution
• Standard shipping shows the longest delivery times
• A small number of products contribute significantly to overall sales
This project helped me strengthen my skills in:
Tableau • Data Visualization • Business Intelligence • Data Analysis • Dashboard Design
